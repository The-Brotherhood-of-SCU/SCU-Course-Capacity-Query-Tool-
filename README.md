# 四川大学课程容量查询工具 (独立版)

SCU URP Course Capacity Query Tool (Standalone)

## 功能简介

本工具用于查询四川大学URP教务系统中各课程的容量信息，包括：
- 课程总容量
- 剩余课余量
- 已选人数
- 课程详细信息

## 项目特点

- **完全独立**：与原项目完全隔离，不依赖任何原项目资源
- **单文件运行**：所有功能整合在一个Python文件中
- **交互式查询**：支持命令行参数和交互式两种模式
- **详细显示**：同时显示课容量、课余量和已选人数

## 安装要求

### 环境要求
- Python 3.7+
- pip 包管理器

### 安装依赖

```bash
pip install -r requirements.txt
```

依赖包列表：
- requests >= 2.31.0
- ddddocr >= 1.4.11
- onnxruntime >= 1.15.0
- pillow >= 10.0.0
- numpy >= 1.24.0

## 使用方法

### 1. 配置账号信息

复制 `config.json.example` 为 `config.json`，并填写你的学号和密码：

```json
{
  "username": "你的学号",
  "password": "你的密码",
  "major_id": "",
  "sleep_time": 2,
  "sleep_jitter": 0.5
}
```

### 2. 运行程序

#### 交互式模式（推荐）

```bash
python course_capacity_query.py
```

进入交互式模式后：
- 输入**课程关键词**查询课程
- 输入 `d+序号`（如 `d1`）查看课程详情
- 输入 `q` 退出程序

#### 命令行直接查询

```bash
python course_capacity_query.py -k "高等数学"
# 或
python course_capacity_query.py --keyword "英语"
```

## 注意事项

1. **账号安全**：`config.json` 包含敏感信息，请勿上传到公共代码仓库
2. **网络环境**：需要在校园网或VPN环境下使用
3. **使用频率**：请勿过于频繁查询，以免对教务系统造成压力
4. **验证码识别**：使用ddddocr自动识别验证码，偶尔可能识别失败，会自动重试

## 与原项目的关系

本项目是从 [SCU-URP-Helper](https://github.com/LittFlower/SCU-URP-Helper) 项目中提取的**独立版本**，仅保留课程容量查询功能：
- 移除了抢课功能
- 移除了教评功能
- 整合为单文件，便于部署和使用
