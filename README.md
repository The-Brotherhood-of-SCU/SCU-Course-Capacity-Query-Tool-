# 四川大学课程容量查询工具 

SCU URP Course Capacity Query Tool 

## 功能简介

本工具用于查询四川大学URP教务系统中各课程的容量信息，包括：
- 课程总容量
- 剩余课余量
- 已选人数
- 课程详细信息

## 项目特点

- **单文件运行**：所有功能整合在一个Python文件中
- **交互式查询**：支持命令行参数和交互式两种模式
- **详细显示**：同时显示课容量、课余量和已选人数

## 安装要求

### 环境要求
- Python 3.7+
- pip 包管理器

### 安装依赖

```bash
pip install -r requirements.txt
```

依赖包列表：
- requests >= 2.31.0
- ddddocr >= 1.4.11
- onnxruntime >= 1.15.0
- pillow >= 10.0.0
- numpy >= 1.24.0

## 使用方法

### 1. 配置账号信息

 在`config.json`中填写你的学号、密码和major_id：

```json
{
  "username": "你的学号",
  "password": "你的密码",
  "major_id": "",
  "sleep_time": 2,
  "sleep_jitter": 0.5
}
```

**major_id获取办法：**进入选课页面，按F12打开开发者工具，点击“网络”，过滤器一栏点击“文档”，会看到index?fajhh=xxxxxxxxxxxx。其中fajhh后面的那串数字就是你的major_id

### 2. 运行程序

#### 交互式模式（推荐）

```bash
python course_capacity_query.py
```

进入交互式模式后：
- 输入**课程关键词**查询课程
- 输入 `d+序号`（如 `d1`）查看课程详情
- 输入 `q` 退出程序

#### 命令行直接查询

```bash
python course_capacity_query.py -k "高等数学"
# 或
python course_capacity_query.py --keyword "英语"
```

## 注意事项

1. **账号安全**：`config.json` 包含敏感信息，请勿上传到公共代码仓库
2. **使用频率**：请勿过于频繁查询，以免对教务系统造成压力
4. **验证码识别**：使用ddddocr自动识别验证码，偶尔可能识别失败，会自动重试
